Solution : Python3 사용
N을 먼저 5로 나누어 그 몫과 나머지를 구한다.
5로 나누어 떨어질 경우 몫을 리턴한다.
5로 나누어 떨어지지 않을 경우, 나머지가 3으로 나누어 떨어질 때까지 5kg짜리 봉지의 개수를 줄인다.
예외를 처리한다.
###
n = int(input())

result = 0
box3 = 0

box5 = n//5
tmp = n%5

if (n%5 == 0):
	box3 = 0

else:
	while(True):
		if tmp%3 == 0:
			box3 = tmp//3
			break
		else:
			box5 -= 1
			tmp += 5


if box5 < 0:
	result = -1

else:
	result = box5 + box3

print(n, '=', ' 5 X', box5, '+ 3 X', box3)
