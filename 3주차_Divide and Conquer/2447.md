## 문제 이해

|생각한 구현 방법|설명|시간 복잡도|
|-|-|-|


## 코드
```
n = int(input())

star = [['*']]

def draw(star):
  nxstar = []
  for i in range(len(star)):
    nxstar.append(star[i]*3)
  for i in range(len(star)):
    nxstar.append(star[i]+ ([' '*len(star[i][0])] * len(star[i])) + star[i])
  for i in range(len(star)):
    nxstar.append(star[i]*3)
  
  return nxstar

while len(star) < n:
  star = draw(star)

for s in star:
  print(''.join(s))

```
