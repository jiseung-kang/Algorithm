# 별 찍기 - 11

## 문제 이해

|생각한 구현 방법|설명|시간 복잡도|
|-|-|-|


## 코드
```
n = int(input())//6

tri = ['  *   ',' * *  ', '***** '] 

def draw(tri):
  tmp = []
  for i in tri:
    tmp.append(' '*(len(tri))+i+' '*(len(tri)))
  for i in tri:
    tmp.append(i+i)
  return tmp

while n > 0:
  tri = draw(tri)
  n = n//2
  
for i in tri:
  print(i)

```
